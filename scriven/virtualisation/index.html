    VMware Workstation 10 on Windows 8.1: Cannot power on virtual machine, not enough physical memory available

    Share article with Radio3

    Tweet This Post

    I was unable to run my vCenter Appliance this morning and was getting the error above. I had been previously running it with no issues and have plenty of host pRAM. To fix the issue and get the VM to power on I had to do the following;

    1. Open Notepad.exe as Administrator

    2. Open the config.ini file located by default at C:\ProgramData\VMware\VMware Workstation.

    3. Add the following line to the end of the file:

      • vmmon.disableHostParameters = “TRUE”

    4. Save the file.

    5. Reboot your host computer.

    6. Run VMware Workstation and boot your VM.
